## Hi there, I'm Shaw 👋

I’m a data science graduate with a background in immunology and genetics, currently focused on building practical skills in data engineering, machine learning and analytics.

I enjoy working across the full data pipeline, from data cleaning and transformation to modelling and delivering insights through clear visualisations. My experience includes Python, SQL and machine learning projects, with growing interest in data engineering workflows (such as pipelines and orchestration tools) and applying data science to real-world domains, especially healthcare and science.

Outside of tech, I enjoy basketball, table tennis, Pokémon and board games. I like thinking in structured, strategic ways, whether that’s solving problems with data or playing competitive games.
